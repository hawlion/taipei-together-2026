# Taipei Together

친구들과 보기 쉬운 타이베이 2박 3일 일정 페이지입니다.

## Local

```bash
npm run dev
```

브라우저에서 `http://localhost:4173` 을 열면 됩니다.

## Build

```bash
npm run build
```

정적 결과물은 `dist/` 에 생성됩니다.

## Render

1. 이 폴더를 GitHub 저장소로 올립니다.
2. Render에서 Static Site 또는 Blueprint로 연결합니다.
3. `render.yaml` 을 사용하면 `dist/` 를 자동 배포할 수 있습니다.
